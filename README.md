# cryptoX
A browser extension which displays the top ten cryptocurrencies according to market cap.

![](https://raw.githubusercontent.com/iSumitBanik/cryptoX/master/files/img/browser-extension.png)

### Technology Stack
1. HTML,
2. Materialize CSS, and
3. JavaScript.
______________________________________________________________________________________________________________________________
### How to Install in Chrome?

#### 1. Visit chrome://extensions
Click on Load Unpacked
![](https://raw.githubusercontent.com/iSumitBanik/cryptoX/master/files/img/step-1.png)

#### 2. Locate the Project Folder and select it

![](https://raw.githubusercontent.com/iSumitBanik/cryptoX/master/files/img/step-2.png)

#### 3. Extension will be added to the list

![](https://raw.githubusercontent.com/iSumitBanik/cryptoX/master/files/img/step-3.png)
